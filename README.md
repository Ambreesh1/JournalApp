# 📓 Journal App (Spring Boot REST API)

## 🚀 Overview
Journal App is a RESTful API built using Spring Boot that allows users to create, read, update, and delete journal entries.

This project demonstrates backend development concepts including REST APIs, CRUD operations, and project structuring.

---

## 🛠️ Tech Stack
- Java
- Spring Boot
- Maven
- REST API
- IntelliJ IDEA

---

## ⚙️ Features
- Create a journal entry ✍️
- View all journal entries 📖
- Update existing entries 🔄
- Delete entries ❌
- Health check API ✅

---

## 📌 API Endpoints

| Method | Endpoint | Description |
|--------|--------|-------------|
| GET | /journal | Get all entries |
| POST | /journal | Create entry |
| PUT | /journal/{id} | Update entry |
| DELETE | /journal/{id} | Delete entry |
